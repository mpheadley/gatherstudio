# Website Copy & Visual Direction — {{CLIENT_NAME}}

> Fill this out AFTER completing `movements.md` and the Design Brief through Phase 2
> (brand identity, photography direction, structural choices).
>
> This is where the Movements Doc becomes actual website copy AND where you decide the
> visual composition of each section. Words and visuals are designed together — the
> photo you choose changes the copy you need, and vice versa.
>
> **Why this step matters:** Without it, you either write copy in the moment during
> the build, or you bolt on images as an afterthought. Both lead to disconnected
> sections where text and visuals compete instead of complement.
>
> **For each section, fill in two things:**
> 1. **Copy** — headlines, body text, CTAs (from Movements Doc)
> 2. **Visual direction** — layout pattern, what the visitor sees, image/icon/video
>    needs, mood, and sourcing plan
>
> **Headline formatting tips:**
> - **Mixed font weights** create emphasis without color: "We Build **Websites** That Work"
>   Use `<strong>` or bold utility classes to emphasize the key word in any headline.
> - **Accent font** (if selected in Phase 2f): mark which word(s) get the handwritten
>   treatment. Best on 1-2 words per page — hero accent + one scroll section.
> - **Color accent**: mark which word(s) use `--primary` or `--secondary` text color.
> - These stack: a word can be bold + accent font + colored. But pick ONE emphasis
>   technique per headline — don't use all three in every heading.
>
> The visual direction should reference your Design Brief Phase 1b (photography)
> and Phase 2 (brand identity) decisions. If you haven't completed those yet, stop
> and do them first — visual composition requires knowing your palette, fonts, and
> what photos you have (or need).

## Problem Statement (after hero)

One line. Names the villain. No body copy. No icon. Large centered type, light background.

**The line:**
*(Pull from Movements Doc — Villain. Write it as a provocation, not a headline. Under 6 words. Do not write this until the client names their villain in conversation — ask first.)*

**Examples by villain type:**
- Object: "Not a catalog." / "Not a template." / "Not the lowest bid."
- Type: "Not the florist with a binder." / "Not the contractor who ghosts."
- Feeling: "Not a transaction." / "Not a price sheet."
- System: "Not the industry standard." / "Not how it's always been done."

**Visual direction:**
- **Layout:** Full-width section, text centered, generous padding
- **Visual element:** None — white space and the words are the design
- **Mood:** Punchy, matter-of-fact, slightly confrontational

> **Pattern note:** After the hero, one line that names the villain in the fewest possible words. No paragraph. The reader finishes the thought in their head. Do not add a supporting paragraph — it dilutes the punch.

**Execution alternatives** (if one-liner alone isn't enough):

| Format | When to use |
|---|---|
| One line, big type | Default — any site |
| Visual contrast (their way vs. yours) | Photo-heavy brands with strong before/after |
| "Most [X] do Y. We don't." | Direct, confident brands |
| Customer quote naming the bad experience | Trust-heavy industries with real testimonials |
| Short paragraph naming the experience | Longer sales pages, high-consideration purchases |

## Checklist

### Copy
- [ ] Hero headline is about the CUSTOMER, not the business
- [ ] Sub-headline names an emotion or transformation
- [ ] Villain statement is ≤6 words and came from the client's own mouth — not invented
- [ ] Pain points are specific to this trade (not generic "we're the best")
- [ ] 3-step plan matches Movements Doc exactly
- [ ] FAQ is ≤4 questions on homepage — only questions that block the primary CTA
- [ ] FAQ answers address real objections (not softball questions)
- [ ] Final CTA section paints success AND names failure stakes
- [ ] Answer-first content block is written (100-200 words, direct answer)
- [ ] All copy reads at 8th grade level (no jargon)
- [ ] Every section traces back to a Movements Doc element
- [ ] "What We Offer" eyebrow replaced with something specific to this client's positioning
- [ ] Testimonials include at least one that matches the primary service being sold on this page
- [ ] Section order is a deliberate choice — not just the default template order. Ask: does the portfolio need to come before the process for this audience? Does trust need to come before services?

### Visual Direction
- [ ] Every section has a visual direction block filled in (not left blank)
- [ ] Layout pattern chosen for each section (no two adjacent sections use the same pattern)
- [ ] Photo needs mapped to Design Brief Phase 1b shot list numbers
- [ ] Icon library chosen if icons are needed (Lucide / Heroicons / other)
- [ ] Mood descriptions are specific (not generic "looks good")
- [ ] Sections that need photos without them have a fallback plan noted
- [ ] Copy length matches the layout — text-heavy sections have room, photo-led sections are concise

### Integration
- [ ] Visual direction references Design Brief Phase 2 decisions (palette, fonts, structural choices)
- [ ] Photography needs from Phase 1b are mapped to specific sections
- [ ] This document saved as `website-copy.md` in the project folder
